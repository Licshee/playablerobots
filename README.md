## Installing
1. "Download as zip"
2. Extract to a new folder your local mods directory (eg on Windows `%USERPROFILE%\Documents\Paradox Interactive\Stellaris\mod\playablerobots`)
3. Copy `playablerobots.mod` to the `mod` parent folder
